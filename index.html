My Scumblr deployment

  • Scumblr is deloyed on top of MESOS
  • The postgreSQL database is in an other machine
  • The redis key is used on a existing Redis instance with unique namespace
  • Sketchy is deloyed on top of MESOS
  • The PostgreSQL database and Redis datastore are located locally in the same MESOS container
